# GT-V1 — 個人決策風格問卷

## 專案概述
純前端個人決策風格問卷網站，部署在 GitHub Pages。陽光明亮 UI 設計，含粒子動畫與角色卡。

## 技術架構
- **單頁應用**：一個 `index.html` 包含所有 HTML、CSS、JS
- **無框架**：原生 JavaScript，無任何外部依賴（僅用 Google Fonts：Nunito、Quicksand、JetBrains Mono）
- **部署**：GitHub Pages（`main` 分支）

## 問卷設計

### 三軸系統
| 軸 | 代碼 | 正端 (+10) | 負端 (-10) |
|---|---|---|---|
| A 做事風格 | G / P / B | 目標導向 (Goal) | 關係導向 (People) |
| B 能量來源 | E / I / M | 外向充電 (Extrovert) | 內向充電 (Introvert) |
| C 內心羅盤 | S / O / A | 自我導向 (Self) | 他人導向 (Others) |

### 分數範圍
- 每軸 -10 到 +10
- `>= 2` 為正端字母，`<= -2` 為負端字母，中間為中間字母 (B/M/A)

### 27 型命名
三軸各 3 種 = 27 種組合（如 GES 開拓者、PIO 奉獻者、BMA 通才等）

### 修仙靈根系統
- **五行靈根**：金（目標導向）、木（關係導向）、火（外向充電）、水（內向充電）、土（自我導向）、月（他人導向）
- **變異靈根**：雷（金+月衝突）、風（木+土衝突）、冰（水+月衝突）
- 每個類型對應 0-3 個靈根 + 可能的變異靈根

### 計分方式
- 30 題，每題 4 選項，每 5 題一頁（共 6 頁）
- 每選項可同時影響 1-3 軸（加減分）
- 原始分 → 依理論最大/最小值正規化到 -10 ~ +10

## UI 特色
- 明亮陽光風格（淺色主題）
- CSS 變數定義在 `:root`，主色 `#F97316`（橙）
- 粒子動畫（6 顆漂浮光球 + 4 顆閃爍星點）
- 結果頁角色卡含：角色 emoji、靈光環、靈根軌道動畫
- 分頁式問卷（每 5 題一頁，含上/下頁導航）

## 檔案結構
```
GT-V1/
├── index.html    # 主頁面（問卷 + 結果頁）
├── README.md     # GitHub 顯示用說明
├── CLAUDE.md     # 本檔案（AI 開發指引）

```

## 開發注意事項
- 所有程式碼都在 `index.html` 中，修改前務必完整閱讀
- CSS 使用 CSS 變數（`--bg`, `--axA` 等），修改配色從 `:root` 改
- 問題資料在 `Q` 陣列（屬性 `q`=題目, `o`=選項, `s`=分數）
- 類型資料在 `T` 物件（屬性 `n`=中文名, `en`=英文名, `d`=描述, `p`=痛點, `v`=印象, `rt`=靈根, `vr`=變異靈根）
- 靈根資訊在 `ROOT_INFO` 和 `VARIANT_INFO`
- 角色 emoji 在 `CHAR` 物件
- 手機優先設計，斷點 500px
